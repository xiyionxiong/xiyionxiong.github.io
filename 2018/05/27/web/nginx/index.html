Mac和centos下nginx启动和配置

Mac 下操作nginx的命令

配置文件路径

/usr/local/etc/nginx/nginx.conf

启动命令

sudo brew services start nginx

重启命令

sudo brew services restart nginx

关闭命令

sudo brew services stop nginx

编辑hosts 文件

sudo vi /etc/hosts
